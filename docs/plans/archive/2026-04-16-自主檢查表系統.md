# 自主檢查表生成系統 — 前端實作計劃

## Context

CNC 工廠行政人員需要一個 Web 應用程式來製作機台「自主檢查表」。系統需支援建立/編輯/預覽/存檔檢查表，量具統一管理，語音輸入，以及匯出為 Excel/PDF/列印模式。

後端實作計劃請參考 `../../backend/docs/plans/2026-04-16-自主檢查表系統.md`

---

## 一、需安裝的套件

```
vue-router           # 路由
vuetify              # UI 框架 (v3)
@mdi/font            # Vuetify icon font
exceljs              # Excel 匯出
jspdf + jspdf-autotable  # PDF 匯出
```

## 三、路由規劃

| 路徑                     | 元件                 | 說明                    |
| ------------------------ | -------------------- | ----------------------- |
| `/`                      | ChecklistListView    | 首頁 — 已存檔檢查表列表 |
| `/checklist/new`         | ChecklistEditView    | 新增檢查表              |
| `/checklist/:id/edit`    | ChecklistEditView    | 編輯檢查表              |
| `/checklist/:id/preview` | ChecklistPreviewView | 預覽模式                |
| `/gauges`                | GaugeManageView      | 量具管理                |

## 四、TypeScript 型別定義

```typescript
interface ChecklistRow {
  id: string
  position: string // 圖面位置（編號）
  gaugeId: string // 量具 ID
  gaugeName: string // 量具名稱（顯示用）
  inspectionItem: string // 檢驗項目
  remark: string // 備註
}

interface Checklist {
  id: string
  name: string // 文件名稱
  rows: ChecklistRow[]
  createdAt: string
  updatedAt: string
}

interface Gauge {
  id: string
  name: string
  createdAt: string
}
```

## 五、各頁面功能細節

### ChecklistListView（首頁）

- Vuetify `v-data-table` 顯示所有已存檔檢查表
- 欄位：序號（自動編號）| 文件名稱 | 存檔時間
- 按存檔時間降序排列
- 點擊列可進入預覽模式
- 右上角「新增文件」按鈕 → 導向 `/checklist/new`
- AppBar 含「量具管理」入口

### ChecklistEditView（新增/編輯）

- 頂部：文件名稱輸入框（預設值：`自主檢查表-YYYY-MM-DD`，可編輯）
- 表格欄位：圖面位置 | 量具 | 檢驗項目 | 備註
  - 圖面位置：文字輸入
  - 量具：`GaugeSelect` 元件（v-autocomplete + 即時新增按鈕）
  - 檢驗項目：`SpeechInputField` 元件（文字框 + 麥克風按鈕）
  - 備註：`SpeechInputField` 元件
- 底部操作列：「新增一列」按鈕、「刪除選中列」按鈕
- 右上角：「儲存」按鈕 → 存檔後導向預覽模式

### ChecklistPreviewView（預覽模式）

- 唯讀表格顯示檢查表內容
- 操作按鈕：「重新編輯」→ 回到編輯頁
- 「匯出」按鈕 → 開啟 ExportDialog 選擇格式

### GaugeManageView（量具管理）

- 量具列表（v-data-table）
- 新增量具（輸入框 + 新增按鈕）
- 刪除量具（含確認對話框）

## 六、語音輸入 (useSpeechRecognition)

- 使用 `webkitSpeechRecognition` / `SpeechRecognition`
- 預設語言：`zh-TW`
- 頁面上方顯示提示：「語音輸入僅支援 Chrome / Edge 瀏覽器」
- 麥克風按鈕切換錄音狀態（錄音中顯示紅色動畫）
- 辨識結果追加到現有文字後方

## 七、匯出功能 (useExport)

- **Excel (.xlsx)**：使用 `exceljs` 套件生成
- **PDF**：使用 `jspdf` + `jspdf-autotable` 生成含中文表格的 PDF
- **網頁列印**：使用 `window.print()` 搭配 `@media print` CSS

## 八、實作順序

### Phase 1 — 基礎建設

1. 安裝套件（vue-router, vuetify, @mdi/font）
2. 設定 Vuetify plugin、Router、更新 main.ts 和 App.vue
3. 建立型別定義 `types/index.ts`
4. 建立 API service layer `services/api.ts`（以 localStorage 模擬）

### Phase 2 — 核心頁面

5. ChecklistListView — 首頁列表
6. ChecklistEditView — 新增/編輯頁面 + ChecklistTable 元件
7. GaugeSelect 元件（下拉選單 + 即時新增）
8. GaugeManageView — 量具管理頁

### Phase 3 — 語音輸入 + 預覽

9. useSpeechRecognition composable
10. SpeechInputField 元件
11. ChecklistPreviewView — 預覽模式

### Phase 4 — 匯出功能

12. 安裝 exceljs, jspdf, jspdf-autotable
13. useExport composable + ExportDialog 元件

> Phase 5（後端整合）請參考 `../../backend/docs/plans/2026-04-16-自主檢查表系統.md`

## 九、驗證方式

1. `npm run dev` 啟動前端開發伺服器
2. 測試完整流程：新增檢查表 → 填寫資料 → 語音輸入 → 儲存 → 預覽 → 匯出
3. 量具管理：新增/刪除量具，確認下拉選單同步更新
4. 檢查表列表：確認排序、點擊進入預覽
5. 匯出：分別測試 Excel、PDF、列印模式
6. 瀏覽器相容性：確認語音提示在非 Chrome/Edge 正確顯示
